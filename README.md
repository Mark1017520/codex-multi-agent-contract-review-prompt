# Codex 多 Agent 合同审查提示词

这是一个用于 Codex 的多 Agent 合同审查提示词模板。它将合同审查拆分为商务条款、知识产权、数据合规、信息安全、责任限制和汇总审查等角色，适合用于合同初审、签署前风险扫描、谈判要点整理和修改建议准备。

## 文件

- `prompt.md`：完整提示词正文。
- `LICENSE`：项目授权协议。
- `.gitignore`：避免把真实合同、输出报告和本地临时文件误提交到仓库。

## 使用方式

1. 打开 `prompt.md`。
2. 将提示词复制到 Codex 中。
3. 按实际项目替换提示词中的占位内容，例如【我方身份】、【合同类型】、【审查目标】和【重点关注事项】。
4. 将待审合同和背景材料放入 Codex 当前工作区，并在提示词中说明文件路径。

## 适用场景

- SaaS 合作合同
- API / SDK 合作合同
- 技术开发合同
- AI 服务合同
- 数据处理协议
- 平台生态合作协议

## 注意事项

该提示词生成的内容应作为初步审查意见，不应直接替代律师或法务负责人的最终法律判断。使用时请避免将含有商业秘密、个人信息或受保密义务约束的合同材料提交到公开仓库。
