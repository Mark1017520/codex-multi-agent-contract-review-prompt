# 贡献指南

欢迎通过 Issue 或 Pull Request 改进本项目，包括补充适用场景、优化提示词结构、完善输出格式或修正文案。

## 贡献授权

提交贡献即表示你确认：

1. 你有权提交该贡献内容；
2. 你的贡献不包含真实合同、商业秘密、个人信息、账号密钥或其他不应公开的信息；
3. 你的贡献内容同意按本项目的 [CC BY 4.0](./LICENSE) 协议授权。

## 内容边界

请不要提交真实合同文本、客户资料、交易背景细节或受保密义务约束的材料。示例内容应使用虚构、脱敏或通用化表达。

## 贡献建议

- 优先保持提示词结构清晰、可复制、可直接使用。
- 新增审查角色或输出字段时，请同步说明适用场景。
- 涉及法律、合规或行业判断时，请避免表述为最终法律意见。
